## PHP WebSocket Server ##

###  Requirements ###

* PHP 7.1 and swoole extension

### Installation ###

* Install php 7.1
* Install extension swoole
* Install composer
* Install sdk ab-ble-gateway-sdk
```
composer install
```
* Run the script 
```
php server.php
```
